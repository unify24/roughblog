# Hiragana Basics

Hiragana (ひらがな) is one of the three Japanese writing systems. It has 46 base characters.

## Vowels

| Character | Romaji |
|-----------|--------|
| あ | a |
| い | i |
| う | u |
| え | e |
| お | o |

## Practice Tips

- Write each character 10 times daily
- Use flashcards to drill recognition
- Read children's books written in Hiragana

```
あいうえお — a i u e o
かきくけこ — ka ki ku ke ko
```
